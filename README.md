# i-game-journal

## 项目简介

`i-game-journal` 是一个用于归档和发布独立 HTML 页面的仓库。这里的每个页面都是一个独立的游戏日志、记录或互动作品，以纯 HTML 形式保存，无需后端支持即可直接浏览。

本仓库不针对任何特定网站或域名，仅作为个人项目存档与分享的容器。

## 目录说明

```
i-game-journal/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面共用的资源文件（图片、样式、脚本等）
└── README.md       # 本文件
```

- `pages/` 目录下每个文件夹代表一个独立项目，内含其对应的 `index.html` 及相关资源。
- `assets/` 存放多个页面可能共用的素材，便于复用与维护。

## 页面归档说明

- 所有页面均为自包含的 HTML 文件，打开即可运行。
- 每个页面记录一个特定的游戏时刻、机制探索或交互实验。
- 页面内容保持原始状态，仅作必要的格式调整与归档整理。
- 不保证所有页面在所有浏览器或设备上完全兼容，但尽量遵循基本 Web 标准。

## 维护说明

- 本仓库不定期更新，新增页面或修正现有内容。
- 欢迎提出建议或报告问题，但请注意本仓库并非面向公众的软件项目，而是个人存档空间。
